<Section icon={Shield} title="Componentes Essenciais para uma Construção de Torre ASMR">

Para construir uma torre estável e agradável, você deve entender a interação entre as teclas estéticas e os blocos estruturais. Muitos itens decorativos, como caminhos de teclado personalizados, não possuem limites nativos de colisão sólida. Se um jogador pisar diretamente em um bloco de teclado sem suporte, ele cairá direto. Portanto, sobrepor plataformas sólidas, invisíveis ou padrão diretamente abaixo de suas teclas personalizadas é a regra fundamental de qualquer layout de sucesso.

Além das plataformas básicas, a incorporação de blocos especializados pode melhorar drasticamente o ritmo de sua subida. Blocos elásticos, plataformas que desaparecem e painéis deslizantes oferecem variedade física que evita que a jogabilidade se torne monótona.

| Nome do Bloco | Tipo de Colisão | Delta de Altura | Função Estrutural |
| :--- | :--- | :--- | :--- |
| Plataforma Base | Sólida, antiderrapante | 0,5 unidades | Fundação principal para todas as teclas personalizadas |
| Cubo Pequeno | Sólido, padrão | 1,0 unidade | Degraus de elevação vertical e vãos |
| Bloco Desaparecendo | Sólido temporário | 0,2 unidades | Caminhos de quebra-cabeça baseados em velocidade |
| Bloco de Dano | Semissólido | 0,5 unidades | Perigos não letais para criar tensão |

<Callout type="warning" title="Aviso de Colisão">
Certifique-se de que suas plataformas de suporte estejam perfeitamente alinhadas com os limites visuais das teclas do teclado. Colisores desalinhados farão com que os jogadores flutuem no ar ou atravessem as bordas, quebrando a ilusão satisfatória.
</Callout>

</Section>

<Section icon={Target} title="Guia de Construção Passo a Passo">

Construir uma obra-prima vertical requer uma abordagem sistemática. Siga estes passos precisos para erguer sua torre do chão, garantindo que cada nível faça a transição de forma suave para o próximo.

<Steps>
  <Step num="1" title="Estabelecer o Lobby e a Fundação">
    Coloque uma plataforma inicial ampla e sólida. Ela serve como a base acústica onde os jogadores podem ajustar suas configurações e testar os sons padrão do teclado antes de subir.
  </Step>
  <Step num="2" title="Camada do Primeiro Caminho Acústico">
    Coloque uma sequência de plataformas padrão e, em seguida, sobreponha-as com os caminhos de teclado escolhidos. Comece com teclas padrão para acostumar os jogadores ao ritmo dos cliques.
  </Step>
  <Step num="3" title="Introduzir Mecânicas de Slime e Elasticidade">
    À medida que a torre sobe, faça a transição para caminhos de slime e blocos elásticos. A mudança no feedback acústico de cliques nítidos para estalos macios mantém a jogabilidade envolvente.
  </Step>
  <Step num="4" title="Alinhar a Subida Vertical">
    Use cubos pequenos para criar pulos verticais. Certifique-se de que cada degrau vertical tenha uma plataforma de suporte correspondente para evitar problemas de colisão durante subidas de alta tensão.
  </Step>
  <Step num="5" title="Finalizar a Zona de Recompensa">
    Crie uma plataforma de vitória espaçosa no topo da sua torre. Preencha-a com recompensas interativas, como mãos de tapa, caixas e efeitos visuais de partículas para celebrar a conclusão.
  </Step>
</Steps>

Manter uma curva de progressão equilibrada é vital. Se o início for muito difícil, os jogadores irão embora; se o final for muito fácil, a vitória parecerá imerecida. Use o modelo de fluxo de progressão abaixo para estruturar a altura e a dificuldade da sua torre.

| Fase | Altitude Alvo | Mistura de Blocos Principal | Classificação de Dificuldade |
| :--- | :--- | :--- | :--- |
| Fase 1: Entrada | 0 - 20 unidades | Teclas padrão, plataformas planas | ★★☆☆☆ (Fácil) |
| Fase 2: Transição | 21 - 50 unidades | Caminhos de slime, vãos mais largos | ★★★☆☆ (Médio) |
| Fase 3: Subida | 51 - 90 unidades | Cubos pequenos, pulos de vidro | ★★★★☆ (Difícil) |
| Fase 4: Topo | Mais de 91 unidades | Degraus que desaparecem, teclas RGB | ★★★★★ (Especialista) |

<Callout type="success" title="Dica de Construção">
Use a ferramenta de movimento preciso (fine-move) para ajustar as distâncias dos pulos. Um espaçamento de 3 a 4 unidades entre os blocos é ideal para a física de pulo padrão do Roblox, garantindo uma corrida suave e ininterrupta.
</Callout>

</Section>
